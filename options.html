Team Search Tool FAQ


Team Search Tool is only available to people within the google.com domain. Users must have prior access to listed systems for the tool to work, this application will not circumvent any security restrictions.

Yes - there are four keyboard shortcuts available when using the dropdown search menu:

ENTER (Predictive Search)
Pressing 'ENTER' will search the appropriate system based on query syntax.

Pressing 'ENTER' when the search field is blank is a shortcut to the Team System 1 Bulk Cable Fibers (OSP & ISP) search.

CTRL+ENTER
Searches Team System 2.

ALT+ENTER
Searches Team System 2 and Team System 1 at the same time.

CTRL+ALT+ENTER
Searches Team System 1 for a list of items separated by commas or spaces.
This will only search for unique values, any duplicates will be ignored.

Team Quick Search will search based on context of highlighted text.

This tool will search Team System 1, Team System 2, Relational Database, or Contracts System depending on the syntax of the selected text.

  • Highlight text you wish to search
  • Right click mouse
  • Click on Team Quick Search

Pressing 'ENTER' on the search bar (Predictive Search) will evaluate the query based on regular expressions (e.g. Contracts System IDs, FQNs, span IDs, OMS IDs) and search the default system for the provided syntax.

Clicking on the buttons is only necessary when you wish to search a system other than the defaults.
